Tier 3: Running Models Locally on Your PC

For the best performance, privacy, and access to the most powerful models (like large DeepSeek, Llama 3, CodeLlama, BitNet variants etc.), you need to run them directly on your own computer using dedicated tools. This webpage cannot run them directly for you.

Recommended Tools:

  • Ollama (Easy CLI/API)
  • LM Studio (GUI)
  • Jan (GUI, Open Source)
  • llama.cpp (Core Engine)
  • Pinokio (Browser for AI apps)

Install one of these, then download models (often in GGUF format) within the tool or from Hugging Face.

Example Models to Try Locally:

(Check compatibility with your chosen tool & hardware)

  • DeepSeek (LLM & Coder versions)
  • Llama 3 (8B, 70B)
  • Mistral / Mixtral
  • Phi-3 (Mini, Small, Medium)
  • CodeLlama

Why Use These Tools?

  • Run much larger models.
  • Better performance (uses your GPU/CPU fully).
  • Offline usage possible.
  • Full data privacy.
  • More customization options.
